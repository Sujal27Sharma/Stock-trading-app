# ⬡ NEXUS Trading Simulator

![Python](https://img.shields.io/badge/Python-3.x-blue.svg)
![Flask](https://img.shields.io/badge/Flask-Framework-green.svg)

NEXUS is a full-stack web application designed to simulate real-time stock market trading. It provides a comprehensive dashboard for tracking Indian market indices, dynamic historical charts, and secure simulated order execution.

## 📖 Project Documentation
To keep this repository organized, the documentation has been split into detailed files:
* **[System Architecture](ARCHITECTURE.md):** Tech stack, design patterns, and folder structure.
* **[Backend Routes](ROUTES.md):** Detailed breakdown of the Flask backend endpoints and trading logic.

## 🚀 Quick Start (Local Setup)
1. Clone this repository: `git clone https://github.com/yourusername/nexus-trading-app.git`
2. Navigate to the directory: `cd nexus-trading-app`
3. Install the required dependency: `pip install flask`
4. Run the application: `python app.py`
5. Open your browser and navigate to `http://127.0.0.1:5000/`

**Note:** You can use any dummy email and password on the login screen for local testing.
